# Ruby Chess

Two players command line chess game written in Ruby.

All the rules of chess are implemented, including en passant, pawn promotion and castling.

It may be run with `ruby play.rb`

## Features

- Load / Save
- En passant
- Castling
- Pawn promotion
- Available destinations
